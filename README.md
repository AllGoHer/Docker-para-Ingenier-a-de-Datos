El presente archivo hace el recorrido sobre el proceso real de de un ingeniero de datos para creación de container en DOCKER, que va desde la creación de una app que conecta a bases de datos hasta su empaquetado.

Abordando los siguientes temas:

* Creación de App de conección de dase de datos.
  
* Dockerfile.
  
* Requerimientos.
  
* Creación de Imagenes.
  
* Capa de lectura de imágenes y capa de escritura de contenedores.
  
* Crear API de Python con Dockerfile.
  
* Asignación de puertos en Docker.
   
* Redes de Docker y microservicios.
  
* Docker Compose para múltiples contenedores.
  
* Volumen de Docker.
